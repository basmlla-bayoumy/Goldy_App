# goldy_app

## Overview

This is a simple Flutter application developed as part of a training program to practice the MVVM (Model-View-ViewModel) architecture.

The app displays the latest prices of gold and silver in a clean and minimal UI.

## Features

* View current gold prices 
* View current silver prices 
* Simple and clean user interface
* Separate screens for each category (Gold & Silver)
* Follows MVVM architecture

## Screens

* **Gold Screen**
  Displays gold prices along with a gold image.

* **Silver Screen**
  Displays silver prices along with a silver image.

## Tech Stack

* Flutter & Dart
* MVVM Architecture
* REST API integration
* State Management (Cubit/Bloc or as implemented)

## Purpose

This project was built to:

* Practice working with MVVM pattern
* Improve API handling skills
* Build clean and maintainable code structure
